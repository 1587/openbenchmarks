00041 # Script for automatically preparing the sources for compilation by
00042 # performing the myriad of necessary steps.  The script attempts to
00043 # detect proper version support, and outputs warnings about particular
00044 # systems that have autotool peculiarities.
00045 #
00046 # Basically, if everything is set up and installed correctly, the
00047 # script will validate that minimum versions of the GNU Build System
00048 # tools are installed, account for several common configuration
00049 # issues, and then simply run autoreconf for you.
00050 #
00051 # If autoreconf fails, which can happen for many valid configurations,
00052 # this script proceeds to run manual preparation steps effectively
00053 # providing a POSIX shell script (mostly complete) reimplementation of
00054 # autoreconf.
00055 #
00056 # The AUTORECONF, AUTOCONF, AUTOMAKE, LIBTOOLIZE, ACLOCAL, AUTOHEADER
00057 # environment variables and corresponding _OPTIONS variables (e.g.
00058 # AUTORECONF_OPTIONS) may be used to override the default automatic
00059 # detection behaviors.  Similarly the _VERSION variables will override
00060 # the minimum required version numbers.
00061 #
00062 # Examples:
00063 #
00064 #   To obtain help on usage:
00065 #     ./autogen.sh --help
00066 #
00067 #   To obtain verbose output:
00068 #     ./autogen.sh --verbose
00069 #
00070 #   To skip autoreconf and prepare manually:
00071 #     AUTORECONF=false ./autogen.sh
